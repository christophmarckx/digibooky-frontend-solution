The ISBN should be Valid before this can work
It should look like this:
978-0-306-40615-7

The first 3 numbers are always 978
The next 9 numbers can be chosen.
The lasts number is a verification number
  • When you multiply the previous numbers by 1 - 3 - 1...
  • Add all these number together
  • Then devide by 10, this gives you a REST
  • Do 10 - REST and you have the last number.

For the given example:
9*1+7*3+8*1+0*3+3*1+0*3+6*1+4*3+0*1+6*3+1*1+5*3
= 93
93 / 10 => 3 REST
10 - 3 = 7
