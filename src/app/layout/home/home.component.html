Stories

Story1: Get all books

Gives the possibility to get an overview of all the available books.

Story2: Show detail of book

Gives the possibility to get one specific book and get more information about this book.

Story3: Search a book by ISBN

Gives the possibility to filter the books, in order to get only the asked books with a specific ISBN-part

Story4: Search a book by title

Gives the possibility to filter the books, in order to get only the asked books with a specific title-part

Story5: Search a book by author

Gives the possibility to filter the books, in order to get only the asked books with a specific author-part

Story6A: Register as member

Gives the possibility to create a new member.

Story7: View members as Admin

Gives the Admin the possibility to view all the existing members.

Story8: Register Admin

Gives the possibility to create new Admins.

Story9: Register Librarian

Gives the possibility to create new Librarians.

Story10A: Register a new book

Gives the possibility to create a new book (By a Librarian).

Story11: Lending a book

Gives the possibility to a member to borrow a book for a specific duration (3 weeks).

Story12: Return a book

Gives the possibility to return a members borrowed book.

Story13: Lent books

Gives the librarian the possibility to get an overview of all the borrowed books.

Story14: Overdue books

Gives the librarian the possibility to get an overview of all the books that are overdue.
